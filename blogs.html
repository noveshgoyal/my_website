useState() in React

Earlier, the major drawback of using the functional component in reactJS was the absence of state, but after hooks came into the action one would definitely choose functional components over class components. A hook is a built-in react function which lets you use the features of react library like state, lifecycle methods and context in functional component. useState is a hook which provides the state to a function, it lets you to reflect the changes in the DOM.
