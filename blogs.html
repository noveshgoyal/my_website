useState() in React

Before hooks(version 16.8), functional components in react were known as stateless component, as there was no way to handle the state in a function, the only way to handle the state was by using class components. But now after the hooks introduction, functions can also have states and lifecycle.

A hook is a function which lets you use state, lifecycle methods and other react features. useState() is one of the most commonly used hooks.

useState(): Also known as state hook, it provides a local state to the functional component. It returns a pair of values in an array, first is the current state value of the state variable and second is a function to set that value. To use the state hook, we need to declare a state variable, which can only be declared inside a function.

Declaring a state variable:

To declare a state variable first we need to import useState from react.

import {UseState} from "react";

After importing we can declare a state variable inside a function.

const [value, setValue] = useState(initial-value)

As discussed earlier, useState returns two values. These values can be assigned by simply assigning a variable to the useState and assigning its first index to one variable and the second index to another elements like, value = arr[0], setValue = arr[1] also. The above used syntax is array destructuring which makes things more easier and readable. Here the only argument to the useState is the initial value which we want to assign to the current state i.e value.

Updating the state:

To update the state we have the setValue function.

setValue( code )

Reading the state:

We can use the variable directly to display the state.

{value}

Example: Consider there is a number and two buttons clicking which will increase the number by one and by two and we want to show this number on the page.

Follow the above steps:
1. Declaring the variable: const [number, setNumber] = useState(0)
2. Update the state: setNumber(number+1), for button one and setNumber(number+2), for button two.
3. use the variable {number} to show the output.
